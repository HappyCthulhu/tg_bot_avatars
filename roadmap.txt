# 0. Цель проекта

Реализовать **Telegram-бота с AI-аватарами**, который:

1. ведёт диалог через LLM
2. хранит **краткосрочную память** (Redis)
3. извлекает **долгосрочные факты** (PostgreSQL)
4. использует **system_prompt аватара**
5. стримит ответ пользователю (editMessageText)
6. извлекает факты **каждые N сообщений**
7. строит prompt из:

```
system_prompt
long-term memory
short-term memory
user message
```
